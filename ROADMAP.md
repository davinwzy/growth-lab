# 项目开发 Roadmap

记录「班级管理系统」从零到一的开发历程。

---

## 🚀 Phase 1: 核心功能搭建

### v0.1 - 基础架构
- [x] 项目初始化 (React + TypeScript + Vite)
- [x] Tailwind CSS 样式系统
- [x] 基础组件库 (Button, Modal)
- [x] Context API 状态管理
- [x] localStorage 数据持久化

### v0.2 - 班级与学生管理
- [x] 多班级创建与切换
- [x] 学生名单管理 (新增/编辑/删除)
- [x] 初次使用引导 (Setup Wizard)

### v0.3 - 分组功能
- [x] 学生分组管理
- [x] 分组颜色自定义
- [x] 随机重新分组
- [x] 小组排序

---

## 🎮 Phase 2: 加分系统与游戏化

### v0.4 - 加分/扣分系统
- [x] 自定义加分项目
- [x] 单个学生加分/扣分
- [x] 小组加分/扣分
- [x] 批量加分操作
- [x] 操作历史记录

### v0.5 - 游戏化系统
- [x] 经验值 (XP) 系统
- [x] 等级系统 (Lv.1 - Lv.10+)
- [x] 等级提升动画与庆祝
- [x] 连续天数 (Streak) 追踪

### v0.6 - 徽章系统
- [x] 内置成就徽章
- [x] 徽章解锁条件
- [x] 徽章收集展示
- [x] 自定义徽章功能
- [x] 徽章编辑与管理

### v0.7 - 奖励兑换
- [x] 自定义奖励项目
- [x] 积分兑换奖励
- [x] 兑换记录追踪

---

## ✅ Phase 3: 出勤系统

### v0.8 - 基础出勤
- [x] 每日出勤签到
- [x] 出勤 +1 分奖励
- [x] 出勤连续天数追踪

### v0.9 - 出勤日历
- [x] 日历视图
- [x] 月份切换导航
- [x] 每日出席率显示
- [x] 出勤历史统计

### v0.10 - 补签与撤销
- [x] 补签功能 (补回过去的出勤)
- [x] 撤销功能 (取消错误的签到)
- [x] 与加分系统整合

---

## 🌐 Phase 4: 多语言与优化

### v0.11 - 国际化
- [x] 中文 / English 双语支持
- [x] 语言切换功能

### v0.12 - 数据管理
- [x] 数据导出 (JSON)
- [x] 数据导入
- [x] 学生头像支持
