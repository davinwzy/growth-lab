# 班级积分系统 - 学生指南 🎮

## 这是什么？

这是一个像游戏一样的班级积分系统！你可以通过上课认真、帮助同学、完成作业等好表现来赚取积分，然后用积分换取各种奖励！

---

## 🎯 RPG 等级系统

就像玩游戏一样，你会有自己的等级！赚取积分的同时，你也会获得 **经验值（XP）**，XP 累积到一定数量就能升级！

### 等级表

| 等级 | 称号 | 需要多少 XP |
|------|------|-------------|
| Lv.1 | 🌱 初心者 (Novice) | 0 XP |
| Lv.2 | 📖 学徒 (Apprentice) | 50 XP |
| Lv.3 | ⚔️ 战士 (Warrior) | 150 XP |
| Lv.4 | 🛡️ 骑士 (Knight) | 350 XP |
| Lv.5 | 🏆 大师 (Master) | 700 XP |
| Lv.6 | 👑 传说 (Legend) | 1200 XP |

**小提示：** 每次获得加分，你同时也会获得相同数量的 XP！比如得到 +3 分，就同时获得 3 XP。

## 👥 组别竞赛

你会被分到一个小组里！小组也有自己的积分！

### 组别积分规则

- 老师可以给整个组加分或减分
- 组别积分和你的个人积分是**分开计算**的
- 每隔一段时间会进行"组别结算"

### 组别结算奖励

结算时，根据组别排名，组内的每个人都能获得额外个人积分：

| 排名 | 每人获得 |
|------|----------|
| 🥇 第一名 | +20 分 |
| 🥈 第二名 | +15 分 |
| 🥉 第三名 | +10 分 |
| 其他组别 | +5 分 |

**小提示：** 和队友一起努力，让你的组成为第一名吧！

## ❓ 常见问题

### Q: 积分和 XP 有什么不同？

**积分（分数）**：可以用来兑换礼物，兑换后会减少
**XP（经验值）**：用来升级，永远不会减少！

举个例子：
- 你有 100 分，兑换了一个 30 分的礼物 → 剩下 70 分
- 但你的 XP 还是 100，不会因为兑换而减少！

### Q: 被扣分会减少 XP 吗？

不会！扣分只会减少你的**可用积分**，你的 XP 和等级都不会受影响。

### Q: 怎样才能快速升级？

1. 每天都努力获得加分，保持连续签到
2. 完成成就可以获得额外奖励分数
3. 帮助小组获胜，结算时能拿到更多分数

### Q: 我可以查看自己的成就吗？

可以！点击你的名字，就能看到你已经解锁了哪些成就徽章，以及还有哪些成就等着你去完成！
