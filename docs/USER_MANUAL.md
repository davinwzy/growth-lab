# 成长实验室（Growth Lab）使用说明书

## 一、系统概览
- 定位：面向教师的游戏化班级管理系统，包含班级、分组、学生、出勤、加减分、成就、奖励与排行榜展示。
- 运行方式：纯前端、离线可用，所有数据保存在浏览器本地。
- 数据安全：不会上传到服务器；清理浏览器数据会导致记录丢失。

## 二、界面入口说明（顶部功能区）
- 班级选择：左侧下拉框可切换班级，旁边可新建班级。
- 主要功能按钮：展示、出勤、学生、组别、成就、礼物、结算、加减分项、历史、设置、语言切换。

## 三、首次使用推荐流程
1. 点击“新建班级”，输入班级名称。
2. 系统自动弹出“班级初始化向导”。
3. 在向导中依次设置组别数量与组名、批量添加学生、分配组别。
4. 进入“加减分项”“成就”“礼物”进行个性化设置。
5. 开始日常使用：出勤签到、加减分记录、展示排行榜。

## 四、功能详解与建议操作

### 1. 班级管理
功能:
- 新建、编辑、删除班级
- 切换当前班级
建议操作:
- 一个学期/一个班级建一个独立班级，避免数据混杂
- 删除班级会同时删除该班级的学生、组别、历史、出勤与成就数据，谨慎操作

### 2. 组别管理
功能:
- 添加/编辑/删除组别
- 快速批量创建组别
- 重新分组（全班学生重新分配）
建议操作:
- 先建组别再添加学生，方便快速归组
- 使用“重新分组”前先完成当天记录，避免打乱统计
- 删除组别会删除该组内学生

### 3. 学生管理
功能:
- 批量添加学生（按行或逗号分隔）
- 编辑学生姓名、组别与头像
- 删除学生
- 一键重置所有学生分数
建议操作:
- 建议使用批量添加提高效率
- 头像只作为识别辅助，不影响统计
- 重置分数只影响学生分数，不影响出勤与历史记录

### 4. 加减分项目管理
功能:
- 自定义加分/减分项目
- 项目分类与排序
建议操作:
- 先配置常用项目，再进行日常加减分
- 负数代表扣分，正数代表加分
- 自定义分类可用于细化“课堂纪律”“作业”“合作”等场景

### 5. 加减分操作
功能:
- 点击学生卡片进行个人加减分
- 点击组别卡片进行组别加减分
- 批量模式进行多人同时加减分
建议操作:
- 个人加减分适用于个体表现
- 组别加减分仅影响组别总分，不影响个人分数
- 批量模式适合全班/小组统一加减分

### 6. 组别结算
功能:
- 根据组别排名给组内每位学生加分
- 结算后组别分数清零
建议操作:
- 建议每周或每月进行一次结算
- 先确认排行榜排序，再执行结算
- 结算记录会写入历史，便于回溯

### 7. 出勤管理
功能:
- 今日签到
- 日历视图（补签、修改、查看比例）
- 出勤统计（按出勤次数、连续天数、姓名排序）
- 免签日设置（包含周末一键设置）
建议操作:
- 当天签到会给学生加 1 分并记经验值
- 补签只补分与经验，不叠加当天连续签到奖励
- 免签日会影响连续签到统计，节假日前建议先设置免签
- 支持撤销出勤，系统会扣除分数并恢复统计

### 8. 游戏化系统（XP/等级/连胜/徽章）
功能:
- 经验值与等级
- 连续得分与连续出勤统计
- 成就徽章解锁与奖励加成
建议操作:
- 正向加分与出勤会增加经验值
- 负向扣分不减少经验值
- 可通过成就的“奖励分数”增强激励效果

等级规则（XP门槛）:
- Lv.1 初心者：0 XP
- Lv.2 学徒：50 XP
- Lv.3 战士：150 XP
- Lv.4 骑士：350 XP
- Lv.5 大师：700 XP
- Lv.6 传说：1200 XP

### 9. 成就管理
功能:
- 自定义成就名称、图标、类别、达成条件、奖励分数
- 支持条件类型如累计XP、连续天数、指定加分项次数、出勤天数等
建议操作:
- 先建立常用加分项，再设置“指定加分项次数”类成就
- 奖励分数可用于特殊激励，但不宜过高

### 10. 奖励管理（兑换商店）
功能:
- 自定义礼物、成本与最低等级要求
- 学生在个人菜单内兑换礼物
建议操作:
- 让奖励与积分节奏匹配，避免“太容易”或“过难”
- 为高等级设置更有吸引力的奖励

### 11. 展示（排行榜/可视化）
功能:
- 组别与个人的周榜/总榜
- 等级分布与班级统计
建议操作:
- 适合课堂展示或活动总结
- 建议固定在周末或月末展示，增强仪式感

### 12. 历史记录
功能:
- 按类型、对象、时间筛选
- 支持撤销单条加减分/兑换/出勤操作
- 支持导出 CSV
- 支持清空历史
建议操作:
- 误操作优先使用“撤销”
- 清空历史只删除记录，不会回滚分数与经验值

### 13. 设置与数据管理
功能:
- 语言切换（中英）
- 导出/导入 JSON
- 数据统计概览
建议操作:
- 定期导出 JSON 进行备份
- 导入会覆盖当前所有数据，建议先备份再导入

## 五、推荐日常使用节奏
- 每天：出勤签到、课堂行为加减分
- 每周：查看周榜、进行组别结算
- 每月：奖励兑换汇总、调整奖励成本与成就条件
- 学期开始：建立班级、组别、学生名单与评分体系
- 学期结束：导出数据保存成果与成长记录

## 六、注意事项
- 删除班级或组别会删除对应数据，无法恢复
- 出勤免签与撤销会重新计算连续统计
- 数据仅保存在当前浏览器与设备中
