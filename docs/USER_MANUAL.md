## 一、系统介绍

### 什么是成长实验室？

成长实验室是一款**游戏化班级管理工具**，帮助老师：

- 📊 记录学生的课堂表现（加分/减分）
- ✅ 管理出勤签到
- 🏆 通过等级、成就、奖励激励学生
- 👥 进行小组竞赛与结算
- 📈 展示排行榜，增强课堂仪式感

### 核心特点

| 特点 | 说明 |
|------|------|
| ☁️ 云端同步 | 用 Google 账户登录，数据实时同步，任何设备都能用 |
| 🎮 游戏化设计 | 经验值、等级、连胜、成就徽章，让学生更有动力 |
| 📱 响应式界面 | 电脑、平板、手机都能用 |
| 🌐 双语支持 | 支持中文和英文切换 |

## 二、开始使用

### 第一步：打开网站

访问：**https://davinwzy.github.io/growth-lab**

### 第二步：登录账户

1. 点击「使用 Google 登录」按钮
2. 选择你的 Google 账户
3. 授权登录

> 💡 首次登录会自动创建账户，无需注册。

### 第三步：选择语言

- 登录页面右上角可切换语言
- 登录后在顶部栏也可以切换

## 三、首次设置流程

建议按以下顺序进行初始设置：

### 1. 创建班级

点击「创建班级」，输入班级名称（如：三年一班）。

### 2. 设置组别

系统会弹出「班级初始化向导」：

1. 设置组别数量（建议 4-6 组）
2. 为每个组别命名和选择颜色

### 3. 添加学生

在向导中批量添加学生：

```
张三
李四
王五
```

或用逗号分隔：`张三, 李四, 王五`

### 4. 分配组别

将学生分配到对应的组别中。

### 5. 自定义设置（可选）

- **加减分项**：自定义加分/扣分的项目和分数
- **成就**：设置学生可解锁的成就徽章
- **奖励**：设置学生可兑换的礼物

## 四、功能详解

### 1. 班级管理

| 操作 | 说明 |
|------|------|
| 新建班级 | 点击班级选择器旁的「+」按钮 |
| 切换班级 | 点击顶部的班级下拉框 |
| 编辑班级 | 在班级下拉框中点击编辑图标 |
| 删除班级 | 在班级下拉框中点击删除图标 |

> ⚠️ 删除班级会同时删除该班级的所有数据（学生、组别、历史、出勤、成就），无法恢复！

---

### 2. 组别管理

点击顶部「组别」按钮进入管理界面。

**功能：**
- 添加/编辑/删除组别
- 调整组别顺序（拖拽排序）
- 快速批量创建组别
- 重新分组（全班学生重新随机分配）

**建议：**
- 先建组别再添加学生
- 使用「重新分组」前先完成当天记录
- 删除组别会同时删除该组内的学生

---

### 3. 学生管理

点击顶部「学生」按钮进入管理界面。

**功能：**
- 批量添加学生（按行或逗号分隔）
- 编辑学生姓名、组别
- 删除学生
- 一键重置所有学生分数

**建议：**
- 使用批量添加提高效率
- 重置分数只影响分数，不影响出勤与历史记录

---

### 4. 加减分操作

这是日常最常用的功能。

**三种操作方式：**

| 方式 | 操作 | 适用场景 |
|------|------|----------|
| 个人加减分 | 点击学生卡片 | 表扬/提醒个别学生 |
| 组别加减分 | 点击组别标题 | 小组竞赛加分 |
| 批量加减分 | 点击「批量操作」 | 全班/多人统一加减分 |

**操作流程：**
1. 点击学生或组别
2. 弹出加减分菜单
3. 选择加分项或扣分项
4. 系统自动记录

---

### 5. 加减分项管理

点击顶部「加减分项」按钮进入管理界面。

**功能：**
- 自定义加分/减分项目
- 设置每个项目的分数
- 项目分类（如：课堂表现、作业、合作等）

**默认项目示例：**

| 项目 | 分数 | 类型 |
|------|------|------|
| 积极发言 | +2 | 加分 |
| 帮助同学 | +3 | 加分 |
| 迟到 | -1 | 扣分 |
| 未交作业 | -2 | 扣分 |

---

### 6. 出勤管理

点击顶部「出勤」按钮进入出勤界面。

**功能：**

| 功能 | 说明 |
|------|------|
| 今日签到 | 勾选出席的学生，点击确认 |
| 日历视图 | 查看/补签/撤销历史出勤 |
| 出勤统计 | 查看出勤次数、连续天数排行 |
| 免签日 | 设置周末、假期等不计入出勤的日子 |

**出勤奖励：**
- 每次签到：+1 分，+5 XP
- 连续签到会累积「出勤连胜」，可解锁相关成就

**补签 vs 当日签到：**
- 当日签到：加分 + 经验 + 连胜统计
- 补签：只加分和经验，不影响连胜

---

### 7. 组别结算

点击顶部「结算」按钮。

**用途：** 根据组别排名，给组内学生发放奖励分数。

**流程：**
1. 查看当前组别排名
2. 设置各名次的奖励分数（如：第1名每人+5分）
3. 点击「结算」
4. 组别分数清零，学生获得奖励

**建议：** 每周或每月进行一次结算，保持竞争新鲜感。

---

### 8. 游戏化系统

系统自动追踪学生的成长数据。

#### 经验值 (XP) 与等级

| 等级 | 名称 | 所需 XP |
|------|------|---------|
| Lv.1 | 初心者 | 0 |
| Lv.2 | 学徒 | 50 |
| Lv.3 | 战士 | 150 |
| Lv.4 | 骑士 | 350 |
| Lv.5 | 大师 | 700 |
| Lv.6 | 传说 | 1200 |

**XP 获取方式：**
- 正向加分：获得等量 XP
- 出勤签到：+5 XP
- 解锁成就：获得成就奖励 XP

> 💡 扣分不会减少 XP，鼓励学生持续进步。

#### 连胜系统

- **得分连胜**：连续多天获得正向加分
- **出勤连胜**：连续多天出勤签到

连胜可用于解锁特定成就。

---

### 9. 成就管理

点击顶部「成就」按钮进入管理界面。

**功能：**
- 查看/编辑/删除成就
- 自定义成就名称、图标、达成条件、奖励

**达成条件类型：**

| 类型 | 说明 | 示例 |
|------|------|------|
| 累计 XP | 总经验值达到指定值 | 获得 100 XP |
| 达到等级 | 等级达到指定级别 | 达到 Lv.3 |
| 连续得分天数 | 连续多天获得加分 | 连续 7 天得分 |
| 连续出勤天数 | 连续多天签到 | 连续 5 天出勤 |
| 累计出勤天数 | 总出勤次数 | 累计出勤 20 天 |
| 指定加分项次数 | 某个加分项的累计次数 | 「积极发言」10 次 |

**查看学生成就：**
点击学生卡片 → 点击「查看成就」

---

### 10. 奖励兑换

点击顶部「礼物」按钮进入管理界面。

**设置奖励：**
- 奖励名称（如：免作业卡、换座位卡）
- 兑换所需分数
- 最低等级要求（可选）

**学生兑换流程：**
1. 点击学生卡片
2. 点击「兑换礼物」
3. 选择要兑换的奖励
4. 确认兑换，扣除对应分数

---

### 11. 展示（排行榜）

点击顶部「展示」按钮。

**展示内容：**
- 组别排行榜（周榜/总榜）
- 个人排行榜（周榜/总榜）
- 等级分布图
- 班级统计数据

**使用场景：**
- 课堂开始/结束时展示
- 周会/月会总结
- 家长会展示

---

### 12. 历史记录

点击顶部「历史」按钮。

**功能：**
- 查看所有操作记录
- 按类型筛选（加分/扣分/兑换/出勤）
- 撤销操作（会回滚分数和数据）
- 导出 CSV
- 清空历史

**撤销操作：**
点击记录旁的撤销按钮，系统会：
- 回滚分数变化
- 回滚经验值变化
- 更新连胜统计

> ⚠️ 清空历史只删除记录，不会回滚已产生的分数和经验值。

---

### 13. 设置

点击顶部「设置」按钮。

**功能：**

| 功能 | 说明 |
|------|------|
| 语言切换 | 中文 / English |
| 导出数据 | 下载 JSON 备份文件 |
| 导入数据 | 从 JSON 文件恢复数据 |
| 数据统计 | 查看班级、学生、记录数量 |

**建议：** 定期导出 JSON 备份，以防万一。

## 六、常见问题

### Q: 换设备后数据还在吗？

**A:** 是的！只要用同一个 Google 账户登录，数据会自动从云端同步。

---

### Q: 可以在手机上使用吗？

**A:** 可以。系统支持响应式设计，手机、平板、电脑都能正常使用。

---

### Q: 多个老师可以共用一个班级吗？

**A:** 目前不支持。每个 Google 账户的数据是独立的。如需共用，可以用同一个 Google 账户登录。

---

### Q: 学生可以登录查看自己的数据吗？

**A:** 目前暂不支持学生登录。这是未来计划开发的功能。

---

### Q: 数据会丢失吗？

**A:**
- 登录后的数据存储在云端，不会因为换设备或清除浏览器缓存而丢失
- 建议定期在「设置」中导出 JSON 备份
